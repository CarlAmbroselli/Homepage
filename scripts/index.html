Scripts

My main machine is a very minimal MacOS host and I usually create ad-hoc digitalocean instances for any type of work and then run tools over X11 and mount directories using sshfs. This increases security since I barely run any code locally on my machine and allows me to use exactly the hardware resources I want for each project. It also forces me to backup all work and make it reproducible since I destroy these droplets once I'm done.

DigitalOcean

  • List droplets Memorizable version of "doctl compute droplet list"
  • Create droplet Memorizable wrapper around "doctl compute droplet create"
  • Remove droplet List all droplets and print the command to delete one
