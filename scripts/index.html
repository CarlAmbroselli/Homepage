My main machine is a very minimal MacOS host and I usually create ad-hoc digitalocean instances for any type of work and then run tools over X11 and mount directories using sshfs. This increases security since I barely run any code locally on my machine and allows me to use exactly the hardware resouces I want for each project. It also forces me to backup all work and make it reproducible since I destroy these droplets once I'm done.
